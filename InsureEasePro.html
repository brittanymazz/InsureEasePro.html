26 themes — 13 dark & 13 colorful light (real color backgrounds, not white).

Two alert types: Badge (red dot — always works) + Browser popup (appears even when app isn't open, requires permission).
Per-task custom alerts: set when adding any task via the "Alert me" dropdown. Badge alerts always show regardless of browser permission.
🛡️ Insurance Tasks — Cancellations, Renewals, Quotes

Add tasks via the topbar. Cancellations auto-calculate 3 reminder dates (1 month, 2.5 weeks, 1 week before). Each step has a color-coded due date chip. Click Draft Email for AI-written cancellation notices. Renewals track 4 steps: Update → Underwriting → Confirm → Documents. Home/Auto Quotes have their own workflows.

🔔 Smart Notifications & Forgotten Tasks

InsureEase scans tasks every 5 minutes. Enable browser notifications in Settings → Notifications. The Forgotten Tasks banner flags tasks with no activity in 5+ days.

🧠 Brain Dump — Sort anything instantly

Type anything on your mind. AI separates it into Tasks, Emails, Notes, and insurance items. Meeting Transcript mode works with Zoom/Teams transcripts.

📋 Meeting Prep, Status Update, Daily Digest, Weekly Review

Meeting Prep has Before/During/After tabs. Status Update pulls your tasks automatically. Daily Digest gives you an AI morning briefing. Weekly Review closes the week with a plan for Monday.

🎨 Themes — 26 distinct palettes

Settings → Appearance. 13 dark themes (violet, magenta, teal, crimson, etc.) and 13 light themes with REAL color backgrounds — peach, pink, lavender, bubblegum, periwinkle, mint, and more. Not white + accent.

Connect Google Drive
To save docs to Google Drive you need a free Google Cloud project. This is a one-time setup.

Steps:
1. Go to console.cloud.google.com
2. Create a project → Enable "Google Drive API" + "Google Docs API"
3. Go to APIs & Services → Credentials → Create OAuth 2.0 Client ID
4. Application type: Web app. Add your app URL as authorized origin
5. Copy the Client ID and paste it below
